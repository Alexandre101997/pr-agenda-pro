# PR Agenda Pro

Sistema web de gerenciamento de agendamentos e tarefas desenvolvido para a empresa PR Personalização Automotiva.

## Organização parceira
PR Personalização Automotiva

## Problema identificado
A empresa realiza os agendamentos manualmente, principalmente por WhatsApp, o que gera:
- falta de organização dos horários
- possibilidade de agendamentos duplicados
- dificuldade no controle dos serviços
- perda de informações de clientes

## Solução proposta
Desenvolvimento de um sistema web para gerenciamento de agendamentos e tarefas.

## Funcionalidades principais
- cadastro e autenticação de usuários
- cadastro de clientes
- criação de agendamentos
- associação de serviços aos clientes
- quadro Kanban
- integração com calendário
